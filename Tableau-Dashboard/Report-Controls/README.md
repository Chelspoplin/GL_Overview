# GL 2024 Overview – Parameter Controls

This dashboard includes a few highlighted controls that let users change what is being shown on the page without needing separate reports.

---

## Account Type (Parameter)

**Options:**  
- **Revenue**
- **Cost**

**What it does:**  
Switches the dashboard between showing revenue-related GL activity vs cost-related GL activity.

**What changes when you use it:**  
- The **GL Amounts by Breakout** chart updates to reflect the selected account type
- KPI totals and Profit by Month will reflect the selected account type context where applicable

Use this to quickly compare which areas are driving revenue vs which areas are driving costs.

---

## Select Breakout (Parameter)

**Options shown in the dropdown:**  
- **Account Name**
- **Description**
- **Department**

**What it does:**  
Controls how the **GL Amounts by Breakout** bar chart is grouped.

**Example:**  
- **Account Name** → groups totals by GL account name  
- **Description** → groups totals by the transaction/account description level  
- **Department** → groups totals by department

Use this to change the level of detail in the breakout chart depending on what you’re investigating.

---
## Go to Details (Navigation Button)

**What it does:**  
Navigates users from the high-level overview page to a more detailed breakdown page.

**Purpose:**  
- Allows deeper investigation of GL activity  
- Provides transaction-level or expanded breakdown detail  
- Supports follow-up analysis without cluttering the overview dashboard  

Use this button when you identify an area in the summary view that requires deeper analysis.

## Summary

These controls work together to let users:
- Filter the dashboard to a specific department
- Switch between revenue vs cost views
- Change how the breakout chart is grouped (account-level, description-level, or department-level)
